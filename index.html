Hungary, Budapest

FPGA based video game, with VGA display controller and keyboard or mouse control

The project presented a look at how to implement a signal inputs VGA and keyboard in a FPGA. Field Programmable Gate Array (FPGA) technology is gaining popularity among Application- Specific Integrated Circuit (ASIC) designers. Ease of development and maintenance makes FPGAs an attractive solution to many speed and efficiency critical applications. The purpose of this project was to explore FPGAs by implementing a Video game on top of a VGA driver. The project was implemented on Xilinx Spartan3E development board using Verilog hardware description language. The project was started by learning Verilog as well as familiarizing with the Spartan3E development board and Xilinx ISE WebPACK design software. A number of simple applications were developed in order to comfortably proceed on to investigation of working principles of a VGA driver. Verilog implementation of the classic Ping-Pong Game was the first major task of the project. Ping-Pong, as it was named, is a two player game, where each player tries to hit a ball towards the opponent. A number of basic features have been implemented.
